## Dependencies
* [curl](http://curl.haxx.se)
* [libxml2](http://www.xmlsoft.org)


## Build and Install
* clone
```shell
    git clone https://github.com/ajiwo/xmdsclient
```
* build
```shell
    cd xmdsclient
    mkdir build
    cd build
    cmake ..
    make
```
* install
```shell
    sudo make install
```
